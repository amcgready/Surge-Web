🎨 How to Customize

1. Quick Theme Change

Edit your .env file:

EMAIL_THEME=colorful # Options: default, dark, minimal, colorful, professional

2. Custom Colors & Branding

Edit backend/email_config.py:

PRIMARY_COLOR = "#ff6b6b" # Your brand color SECONDARY_COLOR = "#4ecdc4" # Secondary brand color LOGO_TEXT = "🚀" # Logo emoji/text HEADER_TITLE = "Welcome to YourApp!" COMPANY_NAME = "Your Company"

3. Preview Changes

After making changes, refresh this page or visit:

http://localhost:5001/api/preview/email?theme=THEME_NAME

4. Test Email Sending

Send a test email with your current configuration:

curl -X POST http://localhost:5001/api/test/send-email \ -H "Content-Type: application/json" \ -d '{"email": "your-email@example.com"}'

5. Production Setup

Configure SMTP settings in your production environment:

MAIL_SERVER=smtp.gmail.com MAIL_USERNAME=your-email@gmail.com MAIL_PASSWORD=your-app-password EMAIL_THEME=professional

Note: Make sure your backend server is running on http://localhost:5001 to see the previews.
